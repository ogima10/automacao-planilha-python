# Automação de Planilha em Python

Projeto simples em Python para leitura de arquivos CSV e geração de relatório automático.

## O que o projeto faz
- Lê uma planilha CSV
- Soma os valores automaticamente
- Gera um relatório no terminal

## Tecnologias
- Python 3

## Como executar
1. Tenha Python instalado
2. Coloque os arquivos na mesma pasta
3. Execute:
   python relatorio.py
